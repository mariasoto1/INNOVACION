Presentación Parte 1

Procedimiento para armar el mueble plegable.

Para armas este mueble , se necesito bastantes máquinas de madera, que nos ayudaron a darle un mejor acabado y a obtener formas que deseabamos. A continuación se mencionara los pasos que se realizaron para el armado de este mueble.

1.- Tenemos que tener las partes de madena completas y de acuerdo a la medida necesaria, que de preferencia sean exactas.

2.- Proseguimos a lijar la madera, hasta que quede bastante liso.

3.- Empezamos con la base de 77x55 al cual se le única por la parte de afuera las maderas con medidas de 77x12 y 60x12. Para esta parte se utiliza el taladro, para entornillar.

4.- Una ves que ya se tiene la superficie de los costados proseguimos a unir por la parte inferior las maderas de 57x10 mediante tornillos y clavos par obtener mayor dureza de unión.

5.- Se continua con colocar las visagras de 2 pulgadas y 3/4 de pulgada , nuevamente utilizando el taladro.

6.- Finalmente se tiene que unir todas las partes, cortando qla pieza que ira de atras del mueble para lograr ser empotrado tuvo q ser cortado a 45°.
